# Terraform Resources

This directory contains all Terraform projects for Autonomys Network infrastructure.

See the [root README](../README.md) for full documentation on setup, usage, and project structure.

## Quick Reference

```sh
# From resources/terraform/
./manage.sh <project> init      # Initialize Terraform
./manage.sh <project> plan      # Generate plan
./manage.sh <project> apply     # Apply changes
```
